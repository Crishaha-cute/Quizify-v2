🧠 QuiziFy

A simple AI-powered quiz generator that creates quizzes dynamically using the Gemini API.

🚀 Features
Generate quizzes using AI (Gemini API)
Dynamic quiz rendering
Results screen with scoring
Simple frontend + API backend
Deployable on Vercel
🧱 Tech Stack
Next.js / React
TypeScript
Gemini API (Google AI)
Vercel (deployment)
